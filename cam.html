Creating a Menu in Batch

@ECHO off

c

cls

:start

ECHO.

ECHO 1. 1

ECHO 2. 2

ECHO 3. 3

ECHO 4. End

set choice=

set /p choice=Type the number to load batch file.

if not '%choice%'=='' set choice=%choice:~0,1%

if '%choice%'=='1' goto 1

if '%choice%'=='2' goto 2

if '%choice%'=='3' goto 3

if '%choice%'=='4' goto 4

ECHO "%choice%" is not available, try again

ECHO.

goto start

:1

ECHO You chose option 1

goto start

:2

ECHO You chose option 2

goto start

:3

ECHO You chose option 3

goto start

Key

ECHO means that anything after it will be displayed back to the user

Changeable variable decided by writer

Basic Command

goto Variable means that the script will move to :Variable
